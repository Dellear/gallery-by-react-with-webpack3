# gallery-by-react-with-webpack3

## 采用webpack3 + es6 + babel + postcss + yarn重塑gallery-by-react项目。

## Live demo
[https://dellear.github.io/gallery-by-react-with-webpack3/](https://dellear.github.io/gallery-by-react-with-webpack3/)


## Local Develop

```bash
yarn
npm start
```

## For Production

```bash
yarn
npm run dist
```
